
# Santander Bootcamp 2023 - Fullstack Java+Angular

Este é um projeto do curso. Fizemos uma página utilizando html, css e javascript. Puxamos uma api externa chamada PokéAPI para a vizualição de 151 Pokemons. Contendo id, nome, type, types e a imagem de cada Pokemon. 

## Execução e instalação

Será necessário ter o Nodejs instalado.

Pode ser aberto apenas 

Vamos utlizar uma biblioteca, após clonar o repositório, é necessário ir ao diretório raiz desse projeto pelo terminal para poder executar os comandos descritos abaixo. 

```
$ npm install -g http-server
```

Próximo passo vamos subir um servidor http

```
$ http-server ./
```
